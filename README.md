WatchLog
========

`WatchLog`是一个云原生容器日志采集工具。你可以使用它来收集`Docker`、`Containerd`的容器日志并发送到集中式日志管理系统中，例如`elasticsearch` `kafka` `redis`等。

版本兼容
========
**Input**

| Service    | Version     |
|------------|-------------|
| Docker     | 推荐 20.x ➕   |
| Containerd | 推荐 1.20.x ➕ |

**Output**

| Service       | Version     |
|---------------|-------------|
| Elasticsearch | 推荐 7.10.x ➕ |
| Kafka         | 推荐 2.x      |
| Redis         | 推荐 6.x      |
